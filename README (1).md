# HablaRápido 🇪🇸

A Telegram bot for learning Spanish at A2 level through speaking practice.

**Flow:** word + explanation → image → TTS question → you reply by voice → transcription → AI feedback → save to Google Sheets

---

## Files

| File | Purpose |
|------|---------|
| `bot.py` | Main bot — run via systemd |
| `generate_images.py` | One-time script to pre-generate all word images |
| `word_packs.json` | Auto-generated by `generate_images.py` — word list with image filenames |
| `images/` | Pre-generated PNG images, one per word |
| `.env` | API keys (not in git) |
| `google-creds.json` | Google service account credentials (not in git) |

## Setup (VM)

### 1. Clone and install

```bash
cd ~
git clone https://github.com/johnaaiton-art/A2_speaker_spanish.git spanish-bot
cd ~/spanish-bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Create `.env`

```bash
cat > .env << 'EOF'
TELEGRAM_TOKEN=your_token_here
DEEPSEEK_API_KEY=your_deepseek_key_here
SHEET_ID=your_google_sheet_id_here
YANDEX_API_KEY=your_yandex_api_key_here
YANDEX_FOLDER_ID=your_yandex_folder_id_here
EOF
```

### 3. Upload `google-creds.json` (from PowerShell on PC)

```powershell
scp -i C:\Users\John\.ssh\id_rsa "C:\Users\John\YandexDisk\Python\innovative vocab\AI magic\yandex\google-creds.json" yc-user@178.154.198.14:~/spanish-bot/google-creds.json
```

### 4. Pre-generate images (run once, ~5-10 mins)

```bash
cd ~/spanish-bot
source venv/bin/activate
python generate_images.py
```

This creates `images/` and `word_packs.json`. Re-run any time you add new words — it skips images that already exist.

### 5. Create systemd service

```bash
sudo nano /etc/systemd/system/spanish-bot.service
```

```ini
[Unit]
Description=HablaRápido Spanish A2 Bot
After=network.target

[Service]
User=yc-user
WorkingDirectory=/home/yc-user/spanish-bot
EnvironmentFile=/home/yc-user/spanish-bot/.env
ExecStart=/home/yc-user/spanish-bot/venv/bin/python bot.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl daemon-reload
sudo systemctl enable spanish-bot.service
sudo systemctl start spanish-bot.service
sudo systemctl status spanish-bot.service
```

## Google Sheets

Uses the same service account as all other bots: `tts-15@anki-332914.iam.gserviceaccount.com`

Two tabs are created automatically:

| Tab | Contents |
|-----|----------|
| `Spanish A2` | Words the user saved with ✅ — date, word, English, theme, example sentence |
| `Custom Packs` | User-uploaded word packs — pack_name, word, english |

## Bot commands

| Command | Action |
|---------|--------|
| `/start` | Welcome message + list of available themes |
| `/nuevas` | Start next theme in rotation |
| `/temas` | List all available themes |
| `/tema comida` | Start a specific theme |

**During a session:**

| Button | Action |
|--------|--------|
| 🆘 No entiendo | Simplifies the question (up to 3 levels of help) |
| 🔊 Repetir | Replays the TTS question |
| ⏭ Otra palabra | Skips to next word |
| ✅ ¡Me sirve! | Saves word to Google Sheets |
| ⏭ Siguiente | Next word without saving |

**To add a custom word pack**, send a `.json` file to the bot:

```json
{
  "cuerpo_humano": [
    {"word": "la cabeza", "english": "head"},
    {"word": "el brazo",  "english": "arm"},
    {"word": "la rodilla","english": "knee"}
  ]
}
```

Custom packs are saved to the `Custom Packs` sheet tab and loaded into memory immediately. Images for custom words are generated on the fly via Yandex Art.

## APIs used

| API | Purpose |
|-----|---------|
| DeepSeek | Word explanations, conversation questions, feedback, help scaffolding |
| Google Cloud TTS (Chirp3-HD `es-US-Chirp3-HD-Aoede`) | Speaks questions and corrected sentences |
| Google Cloud STT | Transcribes user's voice replies |
| Yandex Art | Pre-generates word images (built-in packs) / on-the-fly (custom packs) |
| Google Sheets | Saves vocab words + custom packs |
